# Inflation over time

This is an R script caluclates the US Inflation over time. 

The U.S. Feb has changed their Inflaion policy to a `Flexible Average Inflation
Targeting` policy. 

On August 2020 the Federal Reserve changed their inflation policy. See more [here](https://www.clevelandfed.org/publications/economic-commentary/2021/ec-202109-flexible-average-inflation-targeting-reactions)
